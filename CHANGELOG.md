## [1.0.0] - 2026-04-07

### 新增

- ⚡ 一键生成小红书评论回复功能
- 🎨 5 种回复风格：幽默搞笑、毒舌犀利、暖心共情、知识科普、自定义
- 🔌 支持 OpenAI 兼容 API（OpenAI、DeepSeek、豆包等）
- ⚙️ 可配置 AI 参数：模型、Temperature、Max Tokens
- 📝 生成回复可编辑后使用
- 🧪 内置 API 连接测试
- 🐛 调试面板：实时日志查看与过滤
- 💾 Chrome Storage 本地配置持久化
- 帖子级别与评论级别均支持回复生成
- 弹窗内实时调整风格和参数
